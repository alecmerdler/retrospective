Timezones: JIVE-67873

Problem
  • Event due dates were being shifted ahead by one day for different timezones.
Solution
  • Default due date time was always 11:59pm of the creating user
  • Event due dates were not localized

XSS: CLOUD-13

Problem
  • Document title could contain JavaScript, and executed when rendered in the Popular Content tile
  • Regression I created when trying to get UTF-8 characters to show
Solution
  • Keep the title in raw state until the very last rendering step
  • Escape the malicious characters using Soy template
  • Able to have special characters and no XSS

Grey Videos: JIVE-73765

Problem
  • Updating a document with a video in the content body using the REST API caused the video to be replaced by a greyscale image.
Solution
  • Learned a lot about the REST API and videos in document content body
  • Turned out to be improper use of the API
  • Documentation is important, both for customers and developers
